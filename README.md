# reset-stuck-maestro

This skill automates the resetting of stuck maestro provisiong

## Installation Instructions

```bash
mkdir -p ~/.agents/skills
cd ~/.agents/skills
git clone git@github.com:cstrail/reset-stuck-maestro.git
```

-  "This will work for both Gemini and Kilo from the .agents/skills folder"

## Verify installed
- "In kilo ```/skills``` and should see reset-stuck-maestro
- "In gemini ```/skills list``` and should see reset-stuck-maestro

## Usage
Once enabled, you can trigger the skill by asking to review a Zhone log:
- "Reset stuck maestro 2175551212"
